# Installation and Setup Scripts

This directory contains installation and configuration scripts for different platforms.

## Available Scripts

- **`install_raspberry_pi.sh`** - Complete installation script for Raspberry Pi OS

## Usage

Make scripts executable and run:

```bash
# Raspberry Pi OS installation
chmod +x scripts/install_raspberry_pi.sh
./scripts/install_raspberry_pi.sh
```

Scripts handle:
- System dependencies installation
- Python package installation
- Hardware configuration
- Permission setup
- Service configuration
